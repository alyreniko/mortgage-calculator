### About The Project

**Mortgage Calculator** is a web application built with **Vue.js** designed to help users estimate their monthly mortgage payments. Whether you're a homebuyer planning your budget or a real estate enthusiast, this tool provides a quick and easy way to calculate payments based on the loan amount, interest rate, and loan term.
